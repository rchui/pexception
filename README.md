# pexception

Pretty exception printing.

## Prerequisites

* pipenv
* colorama

```
pip install pipenv
pipenv sync
```

## Development

```
pipenv sync --dev
```

## Running the tests

```
make test
```

## Deployment

```
pip install pexception
```
